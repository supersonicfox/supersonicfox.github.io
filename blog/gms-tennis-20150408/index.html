GMS Tennis: April 8th -- Timing and Targets

Apr 6th, 2015

Blog entries with the title “GMS Tennis” are specifically for the tennis team at Girls Middle School. Others might find the materials interesting too, but this is a different flow from the main blog.

What To Bring

For the first meeting, everyone should try to acquire a racket that they like reasonably. If you don’t have a racket, GMS has some you can try. I also have a couple spares that might have the right size grip.

What We Will Cover

The first meeting will start off with meeting each other, finding the courts, and some quick Tennis-related introductions.

Timing

After that, we will start working through ‘Timing’. Learning timing is a major part of getting good at any sport and is especially true for Tennis. Besides some specific timing things to work on, we will more generally focus on timing tools which you can use to learn and improve many different tennis strokes. If you want a quick overview there is one at (see http://supersonicfox.com/blog/la-ti-da/)
